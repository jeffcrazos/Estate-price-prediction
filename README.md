# Estate-price-prediction

Проект по предсказанию стоимости жилья с помощью линейной регрессии

Проект посвящён прогнозированию цен на дома в Джакарте и Тебете (Индонезия) с помощью методов машинного обучения и статистического анализа. Основная цель — построить модель, которая на основе характеристик недвижимости (площадь, количество комнат и др.) предсказывает рыночную стоимость объекта.
Основные этапы проекта:

Загрузка и предварительная обработка данных

Использовались библиотеки pandas и numpy для работы с табличными данными
Выполнена выборка релевантных признаков: LB (площадь застройки), LT (площадь участка), KT (спальни), KM (ванные), GRS (места в гараже), HARGA (целевая переменная — цена)
Проведена первичная проверка на пропущенные значения и дубликаты
Разведочный анализ данных (EDA)

Построены описательные статистики (describe()) для понимания распределения признаков
Визуализация распределений с помощью matplotlib и seaborn: KDE-графики, boxplot'ы, countplot'ы
Выявлены выбросы по каждому признаку с использованием метода межквартильного размаха (IQR)
Проанализирована корреляция между признаками и целевой переменной через heatmap и pairplot
Проверка классических предпосылок линейной регрессии

Нормальность остатков: тест Жарке-Бера показал отклонение от нормального распределения (p-value = 0.00)
Мультиколлинеарность: расчёт VIF-факторов — все значения < 10, значит, сильной мультиколлинеарности нет
Гомоскедастичность: визуальный анализ остатков и тест Уайта подтвердили отсутствие гетероскедастичности
Частные тесты значимости коэффициентов

Проверка гипотез о значимости каждого предиктора через p-value
Все признаки (LB, LT, KT, KM, GRS) показали статистически значимое влияние на цену (p-value < 0.05)
Построение и обучение модели

Применена LinearRegression из sklearn для построения регрессионной модели
Данные разделены на обучающую и тестовую выборки через train_test_split (60/40)
Получены коэффициенты модели и интерпретированы: например, увеличение площади участка на 1 м² в среднем повышает цену на ~25.7 млн рупий
Оценка качества модели

Основная метрика: коэффициент детерминации R² = 0.6564 (65.64% точности на тестовой выборке)
Дополнительно проанализированы остатки модели и их распределение
Проведён кейс-стади: предсказание стоимости дома для гипотетического покупателя с заданными параметрами
Визуализация результатов

Графики распределения признаков и целевой переменной
Scatter-plot'ы для анализа связи «признак → цена»
Визуализация остатков модели для диагностики качества предсказаний
Итог

Модель линейной регрессии показала умеренную точность (65.64%), что демонстрирует потенциал метода для задач прогнозирования недвижимости. Для улучшения результатов можно:
1) Добавить новые признаки (локация, год постройки, инфраструктура)
2) Применить нормализацию/стандартизацию данных
3) Поэкспериментировать с более сложными моделями (Ridge, Lasso, ансамбли)
Проект показывает, как с помощью EDA и статистической диагностики можно построить интерпретируемую модель и получить ценные инсайты о факторах, влияющих на стоимость жилья.
